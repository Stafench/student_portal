# LAB05 Report

## 1. What does HEAD represent?
HEAD is a pointer that shows the current commit I am working on. Usually, it points to the latest commit of the current branch.

## 2. Under what conditions does a merge conflict occur?
A merge conflict happens when Git cannot automatically combine changes from two branches. This usually occurs when the same lines of a file were changed differently in both branches.

## 3. What is the difference between git clone and git pull?
git clone is used to copy a remote repository to a local machine for the first time. git pull is used later to download and merge new changes from the remote repository into the existing local repository.
